# Project Flow

1. Generate retail datasets using Faker.
2. Store generated CSV files in `data/raw`.
3. Extract the CSV files.
4. Transform and clean the data.
5. Validate the transformed data.
6. Load the data into PostgreSQL.
7. Create SQL analytics views.
8. Connect Power BI to PostgreSQL.
9. Build an interactive dashboard.
10. Push the completed project to GitHub.
